#  DIY R2-D2 Robot

This project is a homemade R2-D2 robot built using Arduino, ESP32, and various sensors.

## 🔧 Features

* Motor control using L293D driver
* ESP32 Bluetooth communication
* PIR sensor for motion detection
* MPU-6050 for orientation
* Future: Face recognition using OpenCV

##  Tech Used

* Arduino Uno / R4
* ESP32
* C++ (Arduino)
* Python (for OpenCV later)

##  Setup

1. Upload Arduino code from `/arduino`
2. Upload ESP32 code from `/esp32`
3. Connect hardware as per diagrams in `/hardware`
